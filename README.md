# DSA With Java 🚀

This repository contains my **personal Data Structures and Algorithms (DSA) journey using Java**.  
All solutions are organized **topic-wise** with clean code and proper structure, making it easy to learn, revise, and prepare for coding interviews.

---

## 📂 Folder Structure

src/dsa/
├── basics/
├── arrays/
├── strings/
├── recursion/
├── sorting/
├── hashing/
├── patterns/
├── leetcode/
└── practice/

- **basics** → Java fundamentals
- **arrays / strings / recursion / sorting / hashing** → Core DSA topics
- **patterns** → Pattern problems
- **leetcode** → LeetCode problem solutions
- **practice** → Rough practice & experiments

---

## 🧠 LeetCode Solutions Naming Convention

All LeetCode solutions follow this format:


### Examples:
LeetCode_001_TwoSum.java
LeetCode_121_BestTimeToBuySellStock.java
LeetCode_206_ReverseLinkedList.java



This makes it easy to search, revise, and track solved problems.
