# DSA With Java 🚀

This repository contains my **personal Data Structures and Algorithms (DSA) journey using Java**.  
All solutions are organized **topic-wise** with clean code and proper structure, making it easy to learn, revise, and prepare for coding interviews.

src/dsa/
├── basics/
